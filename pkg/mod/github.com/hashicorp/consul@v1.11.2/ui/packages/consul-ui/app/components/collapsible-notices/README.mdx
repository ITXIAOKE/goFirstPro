# CollapsibleNotices

Used as a wrapper to collapse the details of `<Notices/>`.

```hbs preview-template
<CollapsibleNotices>
  <Notice
    @type="error"
    role="alert"
  as |notice|>
    <notice.Header>
      <h3>Header</h3>
    </notice.Header>
    <notice.Body>
      <p>
        Body
      </p>
    </notice.Body>
  </Notice>
  <Notice
    @type="info"
  as |notice|>
    <notice.Header>
      <h3>Header</h3>
    </notice.Header>
    <notice.Body>
      <p>
        Body
      </p>
    </notice.Body>
    <notice.Footer>
      <p>
        Footer
      </p>
    </notice.Footer>
  </Notice>
  <Notice
    @type="warning"
  as |notice|>
    <notice.Header>
      <h3>Header</h3>
    </notice.Header>
    <notice.Body>
      <p>
        Body
      </p>
    </notice.Body>
    <notice.Footer>
      <p>
        Footer
      </p>
    </notice.Footer>
  </Notice>
</CollapsibleNotices>

```

## Arguments

No arguments required. Wrap this component around the needed notices.
